# Multi-Job Fit Reference

## Process

1. For each JD, extract required/preferred skills, experience level, education
2. Score resume match per requirement: Strong (3), Partial (2), Weak (1), None (0)
3. Weight: required qualifications 2x, preferred 1x
4. Rank positions by total weighted score

## Output

Table: Rank | Company — Role | Fit Score | Verdict (Strong/Good/Stretch)

For each position: why it fits, gaps to address, application strategy.

## Recommendations

- Apply to strongest fits first
- For partial fits, suggest specific resume tweaks
- If a skill is missing across multiple JDs, flag it as a development priority
- Spend more customization effort on top fits
