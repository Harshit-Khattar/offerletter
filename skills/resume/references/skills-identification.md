# Skills Identification Reference

## Extracting Skills from JDs

**Explicit skills** — directly stated ("Python required", "experience with Docker")
**Implied skills** — inferred from responsibilities ("collaborate with cross-functional teams" → communication, teamwork)

## Categories

- **Communication**: presenting, documenting, explaining technical concepts
- **Leadership**: leading projects, mentoring, taking initiative
- **Collaboration**: cross-functional work, code reviews, pair programming
- **Problem-solving**: debugging, root cause analysis, optimization
- **Adaptability**: learning new tools, pivoting approaches, handling ambiguity

## Prioritization

1. Skills in "Required" section or mentioned 2+ times → must include
2. Skills in "Preferred" section → include if you have them
3. Implied skills → demonstrate through experience bullets, don't just list
