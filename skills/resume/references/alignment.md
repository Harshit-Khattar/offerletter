# JD Alignment Reference

## Process

1. **Deconstruct the JD** — extract: required skills, preferred skills, key responsibilities, company values, team context
2. **Map resume to JD** — for each requirement, find matching resume content. Note gaps
3. **Tailor by section:**
   - **Skills**: reorder to match JD priority. Add missing keywords naturally
   - **Experience**: rewrite bullets to emphasize JD-relevant aspects. Use JD language
   - **Projects**: lead with most relevant project. Add JD keywords to descriptions
   - **Education**: highlight relevant coursework matching JD
4. **Keyword integration** — use exact phrases from JD (ATS matches strings). Target ~60% of top keywords
5. **Skills translation** — convert academic language to industry terms:
   - "Course project" → "Built" / "Developed"
   - "Homework assignment" → never mention
   - "Professor's research" → "Research on [topic]"
   - "Group project" → describe your specific contribution
