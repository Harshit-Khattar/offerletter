# ATS Optimization Reference

## Section Order

Contact → Summary (optional) → Skills → Experience → Projects → Education

## Formatting Rules

**DO:**
- Standard section headings (Education, Experience, Skills, Projects)
- Consistent date format (Month Year - Month Year)
- Simple bullet points
- Standard fonts (Computer Modern, Latin Modern in LaTeX)
- `\pdfgentounicode=1` and `\input{glyphtounicode}` in LaTeX preamble

**DON'T:**
- Tables for layout (use for data only)
- Images, logos, or graphics
- Headers/footers with critical info (ATS may skip them)
- Two-column layouts
- Custom section names ATS won't recognize

## Keyword Strategy

- Target ~60% of top JD keywords
- Use exact JD phrasing (ATS matches strings)
- If JD says "Machine Learning" and "ML", use both
- Place keywords in Skills section AND naturally in experience bullets
- Don't keyword-stuff — must read naturally
