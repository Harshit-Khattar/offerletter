# Bullet Optimization Reference

## Formula

**[Action Verb] + [What] + [Using What] + [Result/Impact]**

Bad: "Responsible for managing the database"
Good: "Optimized database queries using indexing, reducing response time by 45%"

## Quantification

If user says "I don't have numbers", help estimate:
- Scale: users, records, transactions
- Speed: before vs after, % improvement
- Time: hours saved, delivery speed
- Team: people led, mentored
- Use "50K+", "~30%" for approximations

## Action Verbs

Technical: Built, Developed, Implemented, Deployed, Optimized, Automated, Migrated, Integrated, Scaled
Research: Analyzed, Modeled, Validated, Benchmarked, Identified
Leadership: Led, Coordinated, Mentored, Spearheaded
Improvement: Improved, Streamlined, Reduced, Accelerated, Simplified

## Redundancy

- No verb should appear more than twice on the resume
- Each bullet must show a different skill/achievement
- If removing a bullet loses no info, remove it
- Merge two weak bullets into one strong one
